Google Material Design Icon scss CDN.

This project uses rawgit as a cdn for the google-material-design-icons

Include the dists/...icons.css file in your project, and add some markup

(Obviously this solution is not perfectly optimized, each icon is its own SVG image. BuyerBeware!).
